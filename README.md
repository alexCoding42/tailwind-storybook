## Getting Started

First, clone the project then run:

```bash
npm run dev
# or
yarn dev
```

## Run Storybook

```bash
npm run storybook
```

This will serve a local build of Storybook on `localhost:6006`

## Deploy new stories

After adding your stories run

```bash
npm run build-storybook
# and
npm run deploy-storybook
```
